About This Project

This web application was created as part of Advanced GIS (FS26) at ETH Zürich. It visualizes environmental sensor data collected using a SenseBox MCU across two walking routes in Zurich on May 12–13, 2026.

Theme — Microclimate Validation: Sensor readings were collected near the official Flüntern meteorological station (SMA) to validate SenseBox accuracy, and in Zollikon to reveal microclimatic differences across Zurich's urban landscape.

Data collected: Temperature (HDC1080 & BMP280), Humidity, Atmospheric Pressure, Light Intensity, and UV Intensity — all GPS-tagged.

External data: Historical citizen-science measurements from nearby senseBox stations are retrieved from the OpenSenseMap API, covering the same collection period (May 12–13, 2026) for direct comparison.

Created by a student of the Geospatial Engineering department (BAUG), ETH Zürich.
